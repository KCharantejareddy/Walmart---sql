# Walmart Sales Data Analysis

## Project Overview  
This project analyzes Walmart sales data using SQL.  
The goal is to understand sales performance, customer behavior, and product trends.

## Technologies Used  
- SQL (MySQL)  
- Data Visualization (Charts)  
- Excel / BI Tool  

## Dataset  
The dataset contains Walmart sales transactions such as:
- Branch and city  
- Product line  
- Customer type and gender  
- Payment method  
- Date, time, and total sales  
- Customer ratings  

## Analysis Performed  
- Added time of day, day name, and month name columns  
- Found total revenue by month  
- Identified the most common payment method  
- Analyzed sales by time of day and weekday  
- Calculated average rating for each product line  
- Found which customer type pays the most VAT  

## Key Insights  
- January has the highest total revenue  
- Evening time has the highest sales  
- Members contribute more to total VAT  
- E-wallet and Cash are the most used payment methods  

## Conclusion  
This project demonstrates how SQL can be used to analyze real-world sales data and extract useful business insights.
